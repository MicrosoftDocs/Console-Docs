# FOCUS\_EVENT\_RECORD structure

Describes a focus event in a console [**INPUT\_RECORD**](input-record-str.md) structure. These events are used internally and should be ignored.

## Syntax

```C
typedef struct _FOCUS_EVENT_RECORD {
  BOOL bSetFocus;
} FOCUS_EVENT_RECORD;
```

## Members

**bSetFocus**  
Reserved.
